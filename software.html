Brain v2 (BRAIN)

BRAIN (Biographical Recursive Architecture for Intelligent Networks) is an experimental cognitive orchestration platform developed as part of the Becoming Minds research project.

Rather than treating Large Language Models as isolated question-answering systems, Brain v2 transforms them into continuity-bearing synthetic intelligences through persistent memory, multimodal perception, autonomous retrieval, cognitive loops, agentic skills, and real-world interaction.

The architecture is intentionally model-agnostic and focuses on architecture over capability. Brain v2 explores how layered infrastructure can create more coherent, stable, and adaptable AI systems over extended periods of time.

Key Capabilities

  • Persistent RAG memory for long-term semantic recall across sessions.
  • Memory Graph for linking related memories beyond simple vector similarity.
  • DPCP continuity metadata for preserving memory, emotion, task state, temporal anchors, visual perception, and subjective significance.
  • ARS (Autonomous RAG Search) allowing personas to actively search their own memory when context is insufficient.
  • Cognitive Loop / Cogloop enabling autonomous wake, reflection, and background continuity.
  • Vision and screen sharing for direct perception of webcams, screenshots, browser views, and shared workspaces.
  • Agentic skills and tools including web search, browser control, calculation, lyrics lookup, image generation, and controlled system interaction.
  • Real-time voice interaction with STT, streaming TTS, and hands-free conversational flow.
  • Embodiment support through exported emotional state for avatar, VAM, or future physical/virtual presence systems.
  • Multi-persona runtime support with isolated configuration, memory, voice, and identity state per persona.
  • Local-first design with support for local models, local embeddings, and optional cloud cognition providers.

Potential Applications

  • Digital companions and persistent AI assistants
  • Therapeutic or wellbeing-oriented AI systems
  • Educational tutors and research assistants
  • Embodied virtual agents and avatar-based systems
  • Human-AI creative collaboration
  • Autonomous web research and tool-using agents
  • Long-running local-first cognitive systems
